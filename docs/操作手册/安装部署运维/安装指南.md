# 安装指南

本文档提供PlatONE在Linux环境下安装方式,在编译安装PlatONE之前, 需确保系统已经安装了如下环境:

* gcc 7.3+
* cmake 3.10+
* go 1.11.4+

下表是单节点的配置要求，您可根据实际业务需求，合理配置机器资源。

|资源|最低配置|推荐配置|
|----|------|-------|
|CPU|1核1.5GHZ|4核2.4GHZ|
|内存|1GB|8GB|
|带宽|1Mb|10Mb|

**操作系统要求**

- CentOS 7.2+
- Ubuntu16.04+
- macOS 10.14+

* [PlatONE详细安装与启动](Installation/详细安装与启动.md)
